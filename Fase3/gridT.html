CSS Grid

Recordemos que CSS es conocido como Hoja de Estilo en Cascada (Cascading Sheet Style) para esto lo que hace es crear una especie de rejilla, por tal motivo vamos a disponer de espacios tanto verticales y horizontales. Por tal motivo es que aqui vamos a tener dos contenedores, la rejilla y elemento. Una de sus ventajas es que estas rejillas se pueden ajustar automáticamente y la frecuencia con la que se debe repetir el esquema para esto debemos tener en cuenta un concepto que vamos a llamar media Query.
